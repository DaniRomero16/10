Ejercicio 2 CSS

Crear 4 capas (div) donde todas tenga el mismo tamaño (con un width y un height específicos) y que cada una tenga el color de fondo diferente. Cada una deberá contener un párrafo donde el color, el tamaño de fuente, la tipografía y el interlineado también sean diferentes. Establecer el atributo id a cada una de ellas. Además la última capa deberá contener una imagen de fondo alineada al centro (background-position)
